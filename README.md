# System Footprinting & Privacy Dashboard

## Overview
A lightweight, GUI-driven Python application that simulates system reconnaissance phases and privacy defense controls. Built with `tkinter` and Python's standard library, it runs entirely offline and provides real-time visualization of exposure metrics, dynamic risk scoring, and mitigation toggles.

## Features
- **Phase-Based Footprinting:** Step through Network, OS, Service, and Environment enumeration
- **Live Privacy Dashboard:** Dynamic risk score with visual exposure tracking
- **Defense Controls:** Toggle Firewall, Service Hardening, and Privacy Mode
- **Session Management:** Real-time logging, JSON export, and instant reset
- **Zero Dependencies:** Uses only Python standard library modules
- **Fully Offline:** No network calls, external APIs, or privileged commands

## Requirements
- Python 3.8 or higher
- `tkinter` (included with standard Python installations on Windows/macOS/Linux)

## Installation
1. Clone or download this repository:
   ```bash
   git clone Cyber-Range-as-a-Service-System-Footprinting-Privacy-Dashboard
   cd cyber_range_footprint_dashboard

No additional packages required. Skip pip install.
2.Usage
  1.Launch the application
   'python main.py'

  2.Click ▶ Start Footprint to initialize the simulation.
  3.Use ⏭ Next Phase to manually step through each reconnaissance stage, or let it run automatically.
  4.Apply defensive controls from the right panel to mitigate exposure and observe real-time score changes.
  5.Export session logs as JSON or reset to run a new simulation cycle.
